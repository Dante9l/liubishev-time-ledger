# 柳比歇夫时间账本插件设计稿

## 产品定位

一个以手动快录为核心的 Obsidian 时间账本插件，用柳比歇夫计时法记录每日时间花销，并提供 Markdown 导出与 AI 周/月复盘。

## 核心设计

### 1. 快速补录优先

- 默认录入开始时间 + 结束时间。
- 支持 `0910-1040`、`0910`、`90m` 等快捷写法。
- 自动带出上一条记录结束时间。
- 固定分类 + 自由标签，分类支持最近使用优先。

### 2. 数据主存储与展示分离

- 原始记录存储在插件私有数据文件。
- 统计结果由记录实时计算。
- Markdown/Daily Note 作为展示或导出结果，不作为真相来源。

### 3. 统计与复盘

- 提供今日/本周/本月视图。
- 统计分类、标签、总时长和空档。
- AI 第一版只做周/月复盘生成。

## 核心界面

- 快速录入弹窗
- 今日记录侧边栏
- 统计视图（今日 / 本周 / 本月）
- 设置页（分类、导出、AI）

## 数据模型

- `TimeEntry`
- `Category`
- `Settings`

## 技术策略

- 使用 TypeScript + 官方 sample plugin 结构。
- 通过服务层拆分时间解析、统计计算、Markdown 导出、AI 调用。
- 通过 Obsidian `ItemView` 提供列表与统计界面。

## MVP 边界

### 包含

- 快速补录
- 记录 CRUD
- 今日/周/月统计
- Markdown 导出
- AI 周/月复盘

### 不包含

- 自动活动追踪
- 智能分类
- 多端同步增强
- 复杂拖拽时间轴
